# My Portfolio

## Description
A responsive personal portfolio website showcasing my projects and skills as a full-stack web developer.

## Features
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Smooth Navigation** - Seamless scrolling between sections
- **Mobile Menu** - Hamburger menu for easy navigation on small screens
- **Contact Form** - Fully functional contact form with validation
- **Scroll to Top** - Quick navigation button for long pages
- **Modern UI** - Clean and professional design with smooth animations

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries, Animations)
- Vanilla JavaScript (DOM manipulation, Event handling, Form validation)

## How to Use
1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize content with your own information, projects, and skills
4. Update social media links in the contact section
5. Deploy to a hosting service (GitHub Pages, Netlify, Vercel, etc.)

## Customization Guide
- **Change colors**: Edit CSS variables in `css/style.css` (`:root` section)
- **Update content**: Edit text in `index.html`
- **Add projects**: Add more `.project-card` divs in the projects section
- **Add skills**: Add more `.skill-category` divs in the skills section
